Media Queries

Regras CSS que se aplicam em determinados tamanhos de tela.

/* Mobile primeiro */
.box { background: coral; }

@media (min-width: 768px) {
  .box { background: steelblue; }
}

@media (min-width: 1024px) {
  .box { background: seagreen; }
}

Tipografia Fluida

Usando clamp() o texto escala suave entre mínimo e máximo.

Texto fluido!

font-size: clamp(1rem, 4vw, 3rem);

Unidades úteis

  • % — relativo ao pai
  • rem — relativo ao root (html)
  • vw / vh — relativo à janela
  • ch — largura de um caractere
  • clamp(min, val, max) — valor fluido
